# BG Swimming Ring

BepInEx mod for Burglin' Gnomes.

Adds a craftable inflatable swimming ring equipment item. When equipped, it improves water movement and shows a ring visual on the player.

## Features

- Craftable swimming ring item
- Wearable equipment support
- Water movement behavior while equipped
- Embedded model and inventory icon
- Configurable movement and visual settings

## Installation

Put BG-SwimmingRing-Joive.dll into:

Burglin' Gnomes/BepInEx/plugins

## Config

After the first launch, the config file is created in the BepInEx config folder.

Options include movement values, buoyancy, visual offsets, scale, material override, and debug logs.

## Notes

The mod uses embedded assets, so no separate asset files are required for normal installation.
